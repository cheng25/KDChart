## 代码执行逻辑

1. 程序入口：`main.cpp` 中的 `main` 函数
   - 创建 `QApplication` 对象，初始化Qt框架
   - 实例化 `MainWindow` 对象并显示
   - 启动应用程序事件循环

2. 主窗口初始化：`MainWindow` 构造函数
   - 初始化UI组件
   - 创建图表布局和图表对象
   - 从CSV文件加载数据
   - 创建柱状图并设置数据模型
   - 创建和配置四个坐标轴（顶部、左侧、右侧、底部）
   - 设置坐标轴标题和文本属性
   - 将图表添加到坐标平面
